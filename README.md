# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

# 학습 테스트
* String Class  
* Set Collection

# 요구사항
* 1 ~ 9 까지의 서로 다른 수 3자리 만들기
* 스트라이크: 같은 수가 같은 자리
* 볼: 같은 수가 다른 자리
* 낫싱: 같은 수가 전혀 없는 경우
* 다시 시작 기능
* 종료 기능
